◆ getEventSourceFd()

int http::BodyInMemory::getEventSourceFd ( ) const
virtual

Returns the file descriptor for this body, if it has one.

This method tells the ClientHandler if this body is an "active" (event-driven) source or a "passive" (in-memory) one.

Returns
A valid file descriptor (>= 0) if the source needs to be watched by epoll.
-1 if the source is passive (e.g., a std::string) and has no associated file descriptor.
